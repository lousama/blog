Linux下rz/sz命令的简单安装

Posted on 2015-09-11   |   In Linux   |  

说明

在SecureCRT之类的ssh登陆软件中，通过rz/sz命令来上传和下载文件非常的方便，对于大多数Linux系统来说，默认是不带rz/sz命令的，需要手动安装。

  • rz: 弹出对话框，选择文件上传至当前目录
    • rz -y: 弹出对话框，选择文件上传至当前目录，若存在相同文件名直接覆盖
  • sz [file]: 下载文件至设定目录

安装

下载

下载安装包 lrzsz-0.12.20.tar.gz。
失效的话直接进ohse下载。

安装

默认会安装到/usr/local/bin下面，添加软链接

设置

SecureCRT -> Options -> session options -> X/Y/Zmodem，可以设置文件的默认上传和下载目录。
